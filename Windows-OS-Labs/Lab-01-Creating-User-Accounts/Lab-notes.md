# Lab 01 – Creating User Accounts

## Objective
To create and manage user accounts in a Windows environment and understand how user permissions are applied to system resources.

---

## Environment
This lab was performed on a **Windows 7 virtual machine**, due to hardware limitations of the host device.

---

## Tools Used
- Control Panel  
- User Accounts  
- Windows Explorer  
- Folder Properties (Security Tab)  

---

## Activities Performed
- Created a new user account using the Control Panel  
- Logged into the newly created user account  
- Navigated to the user profile directory (`C:\Users\`)  
- Located the created user folder  
- Accessed folder properties to review security permissions  
- Identified which users and groups have access to the directory  

---

## Issues Encountered & Resolution

**Issue:**  
Some lab instructions referenced Windows 10 interfaces, which were not available in the Windows 7 environment.

**Resolution:**  
Equivalent steps were followed using the Control Panel and system tools available in Windows 7.
